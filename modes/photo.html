2: Photo Mode

あなたの写真を、美しい2.5Dポリゴンアートに変換。

Features

Image Upload

任意の画像をアップロードして、リアルタイムでポリゴン化。顔写真、風景、アート作品など、あらゆる画像に対応します。

Audio Reactive Polygons

音楽に合わせてポリゴンが動的に変化。低音で拡大、高音で色が変わるなど、視覚と聴覚の融合を実現します。

Delaunay Triangulation

高度な三角分割アルゴリズムにより、画像の特徴を保ちながら美しい幾何学模様を生成します。

Real-time Control

ポリゴン数、色の強度、エッジの太さなど、すべてのパラメータをMIDIコントローラーでリアルタイム操作可能。

MIDI Controls

CC Number Parameter Description
CC 1 Intensity ポリゴンの動きの激しさ
CC 2 Hue 色相のシフト
CC 3 Speed アニメーション速度
CC 4 FX Amount エフェクトの強度
CC 9 Polygon Density ポリゴンの密度（最大30個）

How to Use

  1. VJアプリを起動し、「Mode 2: Photo Mode」を選択
  2. 「Upload Photo」ボタンをクリックして画像を選択
  3. 画像が自動的にポリゴン化されます
  4. MIDIコントローラーまたはマウスでパラメータを調整
  5. 音楽を再生して、リアルタイムなビジュアルを楽しむ
