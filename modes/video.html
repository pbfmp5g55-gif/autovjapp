5: Video Mode

プロフェッショナルなビデオデッキ機能を搭載。
A/Bミキシング、MIDIトリガー、リアルタイムエフェクトで、本格的なVJパフォーマンスを実現します。

Features

A/B Deck Mixing

2つの独立したビデオデッキ。クロスフェーダーでスムーズにミックス可能。DJミキサーのような操作感を実現。

MIDI Note Trigger

各ビデオクリップをMIDIノートに割り当て。鍵盤やパッドで瞬時に再生開始できます。

Transition Effects

CUT（瞬時切り替え）とFADE（クロスフェード）の2種類のトランジション。音楽に合わせて使い分け可能。

Auto Pilot Mode

ランダムに動画を自動再生。手を離していても、ダイナミックなビジュアルが続きます。

MIDI Controls

CC Number Parameter Description
CC 1 Crossfader A/B デッキのミックス比率
CC 2 Hue Shift 色相の調整
CC 3 Playback Speed 再生速度（0.5x～2.0x）
CC 4 FX Amount エフェクトの強度
CC 7 Contrast コントラスト調整
CC 8 Zoom ズーム倍率
MIDI Note Video Trigger 各ノートに動画を割り当て

How to Use

  1. 「Upload Video」ボタンから動画ファイルをアップロード（mp4, webm対応）
  2. 各動画の「Learn Note」ボタンをクリック
  3. MIDIコントローラーの任意のノートを押して割り当て
  4. ノートを押すと、対応する動画が再生されます
  5. CC1（クロスフェーダー）でA/Bデッキをミックス
  6. 「Auto Pilot」をONにすると、自動で動画が切り替わります
